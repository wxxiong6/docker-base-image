##  php7.2.5

版本：php7.2.5

镜像使用官方镜像制作而成，增加常用的php扩展（yaf、yar、imagick、mcrypt、mongodb、swoole、xdebug）。

### 构建镜像

```shell
./build.sh
```

### 安装镜像

```shell
./run.sh
```
